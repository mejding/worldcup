# Historical International Results

Place a local historical international football results file here:

`data/historical/international_results.csv`

Expected columns:

- `date`
- `home_team`
- `away_team`
- `home_score`
- `away_score`
- `tournament`
- `neutral`

Optional columns such as `city` and `country` are allowed. The app does not download historical data automatically yet.
